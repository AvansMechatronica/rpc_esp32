Welkom bij Avans: ESP32 NodeMCU RPC Library documentatie

Avans: ESP32 NodeMCU RPC Library Dit is een handleiding voor gebruikers die een ESP32 NodeMCU RPC Library willen gebruiken en ontwikkelen in Python. In deze handleiding wordt exact beschreven hoe benodigde software en gereedschap kan worden gebruikt. Er wordt daarbij uitgegaan van een Windows operating systeem met daarop als Integrated Development Environmet(IDE) van pyCharm.
